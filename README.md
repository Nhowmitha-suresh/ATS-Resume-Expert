# ATS-Resume-Expert

AI-powered Applicant Tracking System (ATS) resume analyzer built with **Streamlit** and **Google Gemini**.  
This application evaluates resumes against job descriptions, calculates ATS match percentage, identifies skill gaps, and generates optimized, job-ready resumes.

---

### **Overview**

Applicant Tracking Systems (ATS) are widely used by companies to filter resumes before human review.  
**ATS-Resume-Expert** helps job seekers improve their chances by analyzing resumes using AI and aligning them with job requirements.

---

### **Features**

- ATS match percentage calculation  
- Resume and job description comparison  
- Missing skills and keyword identification  
- AI-powered resume suggestions  
- Job-specific, ATS-optimized resume generation  
- Interactive Streamlit user interface  

---

### **Tech Stack**

- **Language:** Python  
- **Framework:** Streamlit  
- **AI Model:** Google Gemini  
- **Resume Parsing:** PDF text extraction  
- **Environment:** Python Virtual Environment

### **Installation**

### **Installation**

## Step 1: Clone the Repository
git clone https://github.com/your-username/ATS-Resume-Expert.git
cd ATS-Resume-Expert

## Step 2: Create Virtual Environment
python -m venv .venv

## Step 3: Activate Virtual Environment (Windows)
.venv\Scripts\activate

## Step 4: Activate Virtual Environment (macOS / Linux)
source .venv/bin/activate

## Step 5: Install Dependencies
pip install -r requirements.txt

## Step 6: Configure Environment Variables
Create a .env file in the root directory and add:
GOOGLE_API_KEY=your_google_gemini_api_key

Do not commit the .env file to GitHub.

## Step 7: Run the Application
streamlit run app.py

## Step 8: Access the Application
http://localhost:8501

## How It Works
- Upload a resume in PDF format
- Paste the job description
- AI analyzes resume content and job requirements
- ATS match percentage is calculated
- Missing skills and keywords are identified
- Optimized resume is generated
